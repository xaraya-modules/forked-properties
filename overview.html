DateTime Property

Description

The DateTime dataproperty lets you input times displayed as dates and times. It consists of a series of dropdown and a text fields.

The property stores times internally as seconds.

Usage

Standalone

    <xar:data-input type="datetime" [attr1="value1",...] />
    <xar:data-output type="datetime" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['datetime']" [attr1="value1",...] />
    <xar:data-output property="$properties['datetime']" [attr1="value1",...] />

Attributes

  • start_year: integer [default=-5] The first year of the year dropdown. If not specified it is 5 years before the smaller of the current year or the year value of the property.
  • end_year: integer [default=+5] The last year of the year dropdown. If not specified it is 5 years after the larger of the current year or the year value of the property.
  • format: string [default=''] If the value of this attribute is 'ampm' a dropdown is shown with AM or PM to choose from.
  • minutes: boolean [default=true] If false then the input display of minutes is suppressed.
  • seconds: boolean [default=true] If false then the input display of seconds is suppressed.
  • layout: string [default='default'] The layout options all show year, month and day dropdowns. The layouts available are:
    • byminute: show a dropdown of minute increments.
    • byhalfhour: show a dropdown of half hour increments.
    • byhour: show a dropdown of hours.
  • input_type: string [default='dropdown'] The default input_type shows dropdowns for input. The 'calendar' input type shows a calendar for input.
